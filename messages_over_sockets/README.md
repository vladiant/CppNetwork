# Messages over Sockets

## Start
Start server
```bash
./MessagesServer 127.0.0.1 8889
```

Start client
```bash
./MessagesClient 127.0.0.1 8889
```

Start another client
```bash
./MessagesClient 127.0.0.1 8889
```
